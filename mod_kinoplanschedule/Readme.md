# Kinoplan Schedule \(Расписание Киноплан\)

Модуль для Joomla 6, отображает расписание Киноплана из кэша, созданного TASK плагином.\
Необходимы cinema_id и APIkey Киноплана.\
Макет вывода вы можете доработать самостоятельно.
