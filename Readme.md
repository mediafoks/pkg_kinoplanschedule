# Kinoplan Schedule Package

Планировщик задач (WebCron) и модуль для Joomla 6.\
Парсит расписание фильмов Киноплана. Модуль отображает это расписание Киноплана.\
Необходимы cinema_id и APIkey Киноплана.\
Проверено на Joomla 6.1.1

---

Расширение делал исключительно под свои нужды, и не претендует на истину в последней инстанции :), если вам не хватает каких-то полей или функций, придется править код самим или пишите.
